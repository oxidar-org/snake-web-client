# JS + Rust + WebAssembly

This project combines **JavaScript**, **Rust**, and **WebAssembly (Wasm)**. It uses [`wasm-pack`](https://rustwasm.github.io/wasm-pack/) to compile Rust code to WebAssembly, which is then consumed by the JavaScript (React/Vite) layer.

## Requirements

Make sure you have the following tools installed before getting started:

### 1. Install Rust

Follow the official instructions at:

👉 [https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)

After installing, verify Rust is available:

```bash
rustc --version
```

### 2. Install wasm-pack

`wasm-pack` is the official tool for compiling Rust projects to WebAssembly.

Follow the official instructions at:

👉 [https://rustwasm.github.io/wasm-pack/installer](https://rustwasm.github.io/wasm-pack/installer)

After installing, verify `wasm-pack` is available:

```bash
wasm-pack --version
```

### 3. Install Node.js and npm

The JavaScript layer requires Node.js and npm:

- Recommended: install from [https://nodejs.org/](https://nodejs.org/)
- Or use a version manager like [nvm](https://github.com/nvm-sh/nvm)

Verify both are installed:

```bash
node -v
npm -v
```

## Project Structure

```
.
├── rust/               # Rust source code (WASM crate — all game logic lives here)
├── js/                 # Vite + React app (thin JS host layer)
│   └── src/
│       └── wasm/       # Package generated by wasm-pack (do not edit)
├── build.sh            # Script to compile Rust → WebAssembly
```

---

## How to Run the Project

### Linux & macOS

1. Make sure all requirements above are installed.
2. Open a terminal at the project root.
3. Run the build script:

   ```bash
   sh build.sh
   ```

   `build.sh` will:
   - Compile the Rust code in `rust/`.
   - Generate a WebAssembly package.
   - Place the generated files in `js/src/wasm/`.

4. Start the JavaScript dev server:

```bash
cd js
npm install
npm run dev
```

The app is served at `http://localhost:5173` by default.
